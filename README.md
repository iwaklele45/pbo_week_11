
# PBO Missions - PC Builder

A Java-based PC component selection and building simulator that allows users to create custom PC builds by selecting various components.

## Description

This application simulates the process of building a PC by letting users select different components including:

- Motherboard
- CPU
- RAM (up to 4 sticks)
- Storage devices (up to 4 units)
  - SSD
  - HDD
- Peripherals
  - Mouse
  - Keyboard
  - Monitor

## Features
- Interactive menu-driven interface
- Component validation (ensures essential components are installed)
- Real-time price tracking
- Maximum limit enforcement for RAM and storage devices
- Detailed component information display
- Total price calculation

## System Requirements
- Java Development Kit (JDK) 8 or higher
- Java Runtime Environment (JRE)



## How to Run

- Compile the Java files:

```
    javac *.java
```

- Run the Program

```
    java Main
```

## PC Build Requirements
The program enforces the following requirements:
- Must have a motherboard installed
- Must have a CPU installed
- Must have at least one RAM stick
- Maximum 4 RAM sticks
- Maximum 4 storage devices
- Peripherals are optional
